Passing Arguments by Reference

การส่งพารามิเตอร์โดยการอ้างอิงช่วยให้ฟังก์ชันสามารถเข้าถึงและแก้ไขค่าของตัวแปรที่ส่งเข้าไปได้

PHP: สามารถส่งพารามิเตอร์ในการ passed by reference ได้โดยใช้สัญลักษณ์ "&" ตอนประกาศพารามิเตอร์

Example Code: 
function addFive(&$num) {
    $num += 5;
}

$original = 10;
addFive($original);
echo $original;  // Output: 15

Java: ส่งพารามิเตอร์ด้วย passed by reference จะใช้ได้เฉพาะกับ Object แต่ primitive type ส่งจะต้องใช้แบบ Passed By Value แทน

Example Code: 
public class Example {
    public static void addFive(int num) {
        num += 5;
    }

    public static void main(String[] args) {
        int original = 10;
        addFive(original);
        System.out.println(original);  // Output: 10
    }
}

C: ใช้ pointer ในการส่งพารามิเตอร์แบบ passed by reference

Example Code: 
void addFive(int* num) {
    *num += 5;
}

int main() {
    int original = 10;
    addFive(&original);
    printf("%d", original);  // Output: 15
    return 0;
}

Python: ส่ง Object โดยการ passed by reference และใช้ได้กับทุก data tpye

Example Code: 
def add_five(num):
    return num + 5

original = 10
print(add_five(original))  // Output: 15

Variable Number of Arguments

สนับสนุนจำนวนพารามิเตอร์ที่แปรผันช่วยให้ฟังก์ชันมีความยืดหยุ่นมากขึ้น

PHP: สามารถรับจำนวนพารามิเตอร์ที่แปรผันโดยใช้ "..."

Example Code: 
function sumMyNumbers(...$x) {
  $n = 0;
  $len = count($x);
  for($i = 0; $i < $len; $i++) {
    $n += $x[$i];
  }
  return $n;
}

$a = sumMyNumbers(5, 2, 6, 2, 7, 7);
echo $a;

Java: ใช้ "..." แบบเดียวกับ PHP

Example Code: 
public static int sum(int... numbers) {
    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    return sum;
}

public static void main(String[] args) {
    System.out.println(sum(1, 2, 3, 4));  // Output: 10
}

C: ใช้ "va_list" สำหรับจำนวนพารามิเตอร์ที่แปรผัน

Example Code: 
#include <stdarg.h>

int sum(int count, ...) {
    va_list args;
    va_start(args, count);
    int result = 0;
    for (int i = 0; i < count; i++) {
        result += va_arg(args, int);
    }
    va_end(args);
    return result;
}

int main() {
    printf("%d", sum(4, 1, 2, 3, 4));  // Output: 10
    return 0;
}

Python: ใช้ "*args" สำหรับจำนวนพารามิเตอร์ที่แปรผัน

Example Code: 
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4))  // Output: 10

PHP is a Loosely Typed Language

PHP เป็นภาษาที่มีการกำหนดประเภทที่หลวม หมายความว่าตัวแปรไม่ต้องถูกประกาศด้วยประเภทและสามารถเปลี่ยนประเภทได้ตามต้องการ

PHP: ไม่จำเป็นต้องประกาศประเภทตัวแปร และสามารถเปลี่ยนประเภทได้ตามต้องการ

Example Code: 
$variable = 5;        // Integer
$variable = "Hello";  // String
echo $variable;       // Output: Hello

Java: Java เป็นภาษา Strong typed จึงจำเป็นต้องประกาศประเภทตัวแปร

Example Code: 
int num = 5;
num = "Hello";  // Error: Type mismatch

C: C เป็นภาษาที่เป็น strong typed จึงจำเป็นต้องประกาศประเภทตัวแปร

Example Code: 
int num = 5;
num = "Hello";  // Error: Type mismatch

Python: Python มีการกำหนดประเภทแบบ dynamic typed

Example Code: 
num = 5
num = "Hello"
print(num)  // Output: Hello

PHP Return Type Declarations

การประกาศประเภทค่าที่คืนช่วยให้แน่ใจว่าฟังก์ชันคืนค่าประเภทที่ถูกต้อง

PHP: ใช้เพื่อระบุประเภทของค่าที่ฟังก์ชันจะคืนค่า

Example Code: 
function add(int $a, int $b): int {
    return $a + $b;
}

echo add(5, 10);  // Output: 15

Java: มีการประกาศประเภทค่าที่คืนโดยใช้ชื่อประเภทก่อนหน้าชื่อฟังก์ชัน

Example Code: 
public static int add(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    System.out.println(add(5, 10));  // Output: 15
}

C: ต้องระบุประเภทค่าที่คืนในฟังก์ชัน

Example Code: 
int add(int a, int b) {
    return a + b;
}

int main() {
    printf("%d", add(5, 10));  // Output: 15
    return 0;
}

Python: ประกาศประเภทค่าที่คืนได้ด้วย "->"

Example Code: 
def add(a: int, b: int) -> int:
    return a + b

print(add(5, 10))  # Output: 15
