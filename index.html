Passing Arguments by Reference

การส่งอาร์กิวเมนต์โดยอ้างอิงคือการที่ฟังก์ชันสามารถเข้าถึงและปรับเปลี่ยนค่าของตัวแปรที่ส่งเข้าไปได้ โดยใน PHP สามารถทำได้โดยใช้สัญลักษณ์ & ในการประกาศพารามิเตอร์ของฟังก์ชัน

PHP: ใน PHP เราสามารถส่งอาร์กิวเมนต์โดยการอ้างอิงโดยใช้สัญลักษณ์ & เพื่อให้ฟังก์ชันสามารถเปลี่ยนแปลงค่าตัวแปรต้นฉบับได้

ตัวอย่างโค้ด: 
function addFive(&$num) {
    $num += 5;
}

$original = 10;
addFive($original);
echo $original;  // ผลลัพธ์: 15

Java: ใน Java การส่งอาร์กิวเมนต์แบบอ้างอิงจะใช้ได้กับอ็อบเจ็กต์ แต่ประเภทพื้นฐานจะส่งแบบค่าจริง

ตัวอย่างโค้ด: 
public class Example {
    public static void addFive(int num) {
        num += 5;
    }

    public static void main(String[] args) {
        int original = 10;
        addFive(original);
        System.out.println(original);  // ผลลัพธ์: 10
    }
}

C: ใน C สามารถใช้พอยเตอร์ในการส่งแบบอ้างอิง

ตัวอย่างโค้ด: 
void addFive(int* num) {
    *num += 5;
}

int main() {
    int original = 10;
    addFive(&original);
    printf("%d", original);  // ผลลัพธ์: 15
    return 0;
}

Python: Python จะส่งอ็อบเจ็กต์โดยอ้างอิงสำหรับทุกประเภท

ตัวอย่างโค้ด: 
def add_five(num):
    return num + 5

original = 10
print(add_five(original))  // ผลลัพธ์: 15

Variable Number of Arguments

การรองรับจำนวนอาร์กิวเมนต์ที่ไม่แน่นอนนี้ช่วยให้ฟังก์ชันมีความยืดหยุ่นมากขึ้น สามารถรับค่าที่แตกต่างกันได้ตามที่ต้องการ

PHP: ใน PHP เราสามารถส่งจำนวนอาร์กิวเมนต์ที่ไม่จำกัดได้โดยใช้ ...

ตัวอย่างโค้ด: 
function sum(...$numbers) {
    return array_sum($numbers);
}

echo sum(1, 2, 3, 4);  // ผลลัพธ์: 10

Java: คล้ายกับ PHP, Java ใช้ ... (varargs)

ตัวอย่างโค้ด: 
public static int sum(int... numbers) {
    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    return sum;
}

public static void main(String[] args) {
    System.out.println(sum(1, 2, 3, 4));  // ผลลัพธ์: 10
}

C: ใน C สามารถใช้ va_list สำหรับอาร์กิวเมนต์ที่มีจำนวนไม่แน่นอน

ตัวอย่างโค้ด: 
#include <stdarg.h>

int sum(int count, ...) {
    va_list args;
    va_start(args, count);
    int result = 0;
    for (int i = 0; i < count; i++) {
        result += va_arg(args, int);
    }
    va_end(args);
    return result;
}

int main() {
    printf("%d", sum(4, 1, 2, 3, 4));  // ผลลัพธ์: 10
    return 0;
}

Python: Python ใช้ *args สำหรับอาร์กิวเมนต์ที่มีจำนวนไม่แน่นอน

ตัวอย่างโค้ด: 
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4))  // ผลลัพธ์: 10

PHP is a Loosely Typed Language

PHP เป็นภาษาที่ไม่ต้องกำหนดประเภทตัวแปร ซึ่งตัวแปรสามารถเปลี่ยนประเภทได้ตามค่าที่กำหนดใหม่

PHP: ใน PHP คุณไม่จำเป็นต้องประกาศประเภทของตัวแปร และสามารถเปลี่ยนประเภทได้ตามต้องการ

ตัวอย่างโค้ด: 
$variable = 5;        // จำนวนเต็ม
$variable = "Hello";  // สตริง
echo $variable;       // ผลลัพธ์: Hello

Java: Java เป็นภาษาที่มีการกำหนดประเภทที่เข้มงวด จำเป็นต้องประกาศประเภทตัวแปร

ตัวอย่างโค้ด: 
int num = 5;
num = "Hello";  // ข้อผิดพลาด: ประเภทไม่ตรงกัน

C: คล้ายกับ Java, C ก็เป็นภาษาที่มีการกำหนดประเภทที่เข้มงวดเช่นกัน

ตัวอย่างโค้ด: 
int num = 5;
num = "Hello";  // ข้อผิดพลาด: ประเภทไม่ตรงกัน

Python: Python เป็นภาษาที่มีการกำหนดประเภทแบบไดนามิก แต่ยังคงเป็นประเภทที่เข้มงวด

ตัวอย่างโค้ด: 
num = 5
num = "Hello"
print(num)  // ผลลัพธ์: Hello

PHP Return Type Declarations

การประกาศประเภทการคืนค่าให้เราสามารถกำหนดประเภทค่าที่ฟังก์ชันจะส่งกลับได้ ช่วยให้โค้ดมีความชัดเจนและลดข้อผิดพลาด

PHP: ตั้งแต่ PHP 7.0 เป็นต้นไป คุณสามารถประกาศประเภทการคืนค่าในฟังก์ชันได้

ตัวอย่างโค้ด: 
function addNumbers(int $a, int $b): int {
    return $a + $b;
}

echo addNumbers(2, 3);  // ผลลัพธ์: 5

Java: Java จำเป็นต้องประกาศประเภทการคืนค่าในทุกเมธอด

ตัวอย่างโค้ด: 
public static int addNumbers(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    System.out.println(addNumbers(2, 3));  // ผลลัพธ์: 5
}

C: C จำเป็นต้องประกาศประเภทการคืนค่าในทุกฟังก์ชัน

ตัวอย่างโค้ด: 
int addNumbers(int a, int b) {
    return a + b;
}

int main() {
    printf("%d", addNumbers(2, 3));  // ผลลัพธ์: 5
    return 0;
}

Python: Python รองรับการระบุประเภทการคืนค่า แต่ไม่ได้บังคับใช้อย่างเข้มงวด

ตัวอย่างโค้ด: 
def add_numbers(a: int, b: int) -> int:
    return a + b

print(add_numbers(2, 3))  // ผลลัพธ์: 5
