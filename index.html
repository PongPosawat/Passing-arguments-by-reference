การเปรียบเทียบการเขียนโปรแกรม OOP ใน Java, C++, Python

I. ผลรันของโปรแกรมในแต่ละภาษา

Java

// Class Animal
class Animal {
    public Animal() {
        System.out.println("create animal");
    }

    public void sound() {
        System.out.println("animal makes a sound");
    }

    protected void finalize() {
        System.out.println("delete animal");
    }
}

// Class Dog extends Animal
class Dog extends Animal {
    public Dog() {
        System.out.println("dog");
    }

    @Override
    public void sound() {
        System.out.println("dog barks");
    }

    protected void finalize() {
        System.out.println("delete dog");
    }
}

// Main Class
public class Main {
    public static void main(String[] args) {
        Animal a = new Animal();
        Dog d = new Dog();
        a.sound();
        d.sound();
    }
}

C++

#include <iostream>
using namespace std;

// Class Animal
class Animal {
    public:
        Animal() { cout << "create animal" << endl; }
        virtual ~Animal() { cout << "delete animal" << endl; }
        virtual void sound() { cout << "animal makes a sound" << endl; }
};

// Class Dog extends Animal
class Dog : public Animal {
    public:
        Dog() { cout << "dog" << endl; }
        ~Dog() { cout << "delete dog" << endl; }
        void sound() override { cout << "dog barks" << endl; }
};

// Main function
int main() {
    Animal* a = new Animal();
    Dog* d = new Dog();
    a->sound();
    d->sound();
    delete a;
    delete d;
    return 0;
}

Python

# Class Animal
class Animal:
    def __init__(self):
        print("create animal")

    def sound(self):
        print("animal makes a sound")

    def __del__(self):
        print("delete animal")

# Class Dog extends Animal
class Dog(Animal):
    def __init__(self):
        super().__init__()
        print("dog")

    def sound(self):
        print("dog barks")

    def __del__(self):
        print("delete dog")

# Main function
a = Animal()
d = Dog()
a.sound()
d.sound()

II. แนวคิด OOP และการเปรียบเทียบกับโค้ด

Class

Class คือแม่แบบของออบเจ็กต์ ตัวอย่างได้แก่ `Animal`, `Dog`, `Cat`

Object, Instance

Object หรือ Instance คือการสร้างตัวตนจาก Class เช่น ออบเจ็กต์ `a`, `d`, และ `c` ที่สร้างจากคลาส `Animal`, `Dog`, และ `Cat` ตามลำดับ

Subclass, Derived Class

Subclass คือคลาสที่สืบทอดจากคลาสแม่ เช่น `Dog` และ `Cat` ที่สืบทอดจาก `Animal`

Message

Message คือการส่งคำสั่งไปยังออบเจ็กต์ เช่น `a.sound()`, `d.sound()`

Inheritance

Inheritance คือการสืบทอดคุณสมบัติจากคลาสแม่ไปยังคลาสลูก เช่น `Dog` และ `Cat` สืบทอดจาก `Animal`

Polymorphism

Polymorphism คือความสามารถที่ method เดียวกันสามารถทำงานต่างกันใน subclass ต่างๆ เช่น `sound()` ใน `Dog` และ `Cat`

III. Abstract Class

Abstract class คือคลาสที่ไม่สามารถสร้างออบเจ็กต์ได้โดยตรง และมี method ที่ต้อง override ใน subclass ตัวอย่างเช่นการสร้าง abstract class ใน Java, C++, และ Python
