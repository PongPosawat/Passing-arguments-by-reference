Passing Arguments by Reference

Passing arguments by reference allows a function to access and modify the value of variables passed to it. In PHP, this is done using the & symbol in the function parameter declaration.

PHP: In PHP, you can pass arguments by reference using the & symbol, allowing the function to modify the original variable.

Example Code: 
function addFive(&$num) {
    $num += 5;
}

$original = 10;
addFive($original);
echo $original;  // Output: 15

Java: In Java, passing by reference is applicable to objects, while primitive types are passed by value.

Example Code: 
public class Example {
    public static void addFive(int num) {
        num += 5;
    }

    public static void main(String[] args) {
        int original = 10;
        addFive(original);
        System.out.println(original);  // Output: 10
    }
}

C: In C, pointers are used for passing by reference.

Example Code: 
void addFive(int* num) {
    *num += 5;
}

int main() {
    int original = 10;
    addFive(&original);
    printf("%d", original);  // Output: 15
    return 0;
}

Python: Python passes objects by reference for all types.

Example Code: 
def add_five(num):
    return num + 5

original = 10
print(add_five(original))  // Output: 15

Variable Number of Arguments

Support for a variable number of arguments allows functions to be more flexible, accepting different amounts of input as needed.

PHP: In PHP, you can accept a variable number of arguments using ....

Example Code: 
function sum(...$numbers) {
    return array_sum($numbers);
}

echo sum(1, 2, 3, 4);  // Output: 10

Java: Similar to PHP, Java uses ... (varargs).

Example Code: 
public static int sum(int... numbers) {
    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    return sum;
}

public static void main(String[] args) {
    System.out.println(sum(1, 2, 3, 4));  // Output: 10
}

C: In C, you can use va_list for a variable number of arguments.

Example Code: 
#include <stdarg.h>

int sum(int count, ...) {
    va_list args;
    va_start(args, count);
    int result = 0;
    for (int i = 0; i < count; i++) {
        result += va_arg(args, int);
    }
    va_end(args);
    return result;
}

int main() {
    printf("%d", sum(4, 1, 2, 3, 4));  // Output: 10
    return 0;
}

Python: Python uses *args for a variable number of arguments.

Example Code: 
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4))  // Output: 10

PHP is a Loosely Typed Language

PHP is a loosely typed language, meaning variables do not have to be declared with a type, and they can change types as needed.

PHP: In PHP, you do not need to declare variable types, and you can change types as desired.

Example Code: 
$variable = 5;        // Integer
$variable = "Hello";  // String
echo $variable;       // Output: Hello

Java: Java is a strongly typed language requiring variable type declarations.

Example Code: 
int num = 5;
num = "Hello";  // Error: Type mismatch

C: C is also a strongly typed language.

Example Code: 
int num = 5;
num = "Hello";  // Error: Type mismatch

Python: Python is dynamically typed, but it is still strongly typed.

Example Code: 
num = 5
num = "Hello"
print(num)  // Output: Hello

PHP Return Type Declarations

Return type declarations allow us to specify the type of value a function will return, improving code clarity and reducing errors.

PHP: Since PHP 7.0, you can declare return types in functions.

Example Code: 
function addNumbers(int $a, int $b): int {
    return $a + $b;
}

echo addNumbers(2, 3);  // Output: 5

Java: Java requires return type declarations for every method.

Example Code: 
public static int addNumbers(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    System.out.println(addNumbers(2, 3));  // Output: 5
}

C: C requires return type declarations for every function.

Example Code: 
int addNumbers(int a, int b) {
    return a + b;
}

int main() {
    printf("%d", addNumbers(2, 3));  // Output: 5
    return 0;
}

Python: Python supports type hints for return types but does not enforce them strictly.

Example Code: 
def add_numbers(a: int, b: int) -> int:
    return a + b

print(add_numbers(2, 3))  // Output: 5
